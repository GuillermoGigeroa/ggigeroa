🖨️ Sistema Impresora

Panel de Control - API REST Completo

ℹ️ Esta interfaz web comunica directamente con los endpoints RESTful en /api/

✓ Validación del Sistema

Valida el estado actual del servidor y obtiene información de la aplicación.

📥 Consultar Datos (GET)

Obtiene datos de un recurso específico mediante su ID.

📤 Procesar/Crear Datos (POST)

Envía datos al servidor para procesar o crear nuevos registros.

🔄 Actualizar Datos (PUT)

Modifica los datos de un recurso existente.

🗑️ Eliminar Datos (DELETE)

Elimina un recurso existente del servidor.

⚠️ Esta operación es destructiva y no se puede deshacer.
