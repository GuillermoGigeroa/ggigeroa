🚀 Inicio Rápido

Requisitos

  • Java 17+
  • Maven 3.8+ (incluido: mvnw)
  • MySQL 5.7+ (opcional)

1. Compilar

mvnw.cmd clean package -DskipTests

2. Ejecutar

Con Maven: 
mvnw.cmd spring-boot:run
Con JAR: 
java -jar target/impresora.ggigeroa-0.0.1-SNAPSHOT.jar

3. Acceder

La aplicación se levanta en http://localhost:8080
