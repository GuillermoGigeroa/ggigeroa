---
description: Cómo reiniciar el servidor Spring Boot localmente parando el proceso en curso
---

Para reiniciar el servidor web de Spring Boot, liberando el puerto 8080 y levantando una instancia nueva, debes invocar el script de PowerShell de la raíz del proyecto.

1. Ejecuta el script de reinicio en Foreground:
// turbo
```powershell
.\restart-server.ps1
```
